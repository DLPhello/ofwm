# Design reference

- **Source:** Live site at oatleywines.com.au (structure + imagery only - not a layout clone).
- **Direction:** Warm Australian luxury trade - paper grain, ink + wine + bronze, Cormorant display (light weights), crossfade hero with slow drift (respects reduced motion), scroll read-progress, gallery tiles with bronze hover wash, featured news, page enter fade, editorial About (drop cap + pull quote).
- **Motion:** One easing curve `cubic-bezier(0.22, 1, 0.36, 1)`; no gratuitous loops beyond hero.
- **Screenshots:** Optional - capture hero + portfolio grid + news list in browser for future polish.
